Deskripsi app (MyTaskList adalah aplikasi manajemen tugas berbasis mobile yang dibangun menggunakan framework React Native untuk membantu pengguna mengelola produktivitas harian secara efisien. Aplikasi ini mengintegrasikan fitur CRUD yang memungkinkan pengguna menambah, menandai tugas yang selesai, serta menghapus tugas dengan antarmuka dark mode yang modern dan responsif. Dilengkapi dengan sistem validasi input dan indikator prioritas visual, aplikasi ini memastikan pengelolaan daftar tugas menjadi lebih terstruktur dan informatif.)
Fitur yang diimplementasikan 
[1] Setup & Running HP Fisik: Project dibuat dengan npx create-expo-app dan diuji langsung melalui Expo Go.

[2] Komponen Dasar (P02 & P03): Menggunakan View, Text, TouchableOpacity, SafeAreaView, dan StatusBar.

[3] Layouting: Implementasi StyleSheet.create dengan sistem Flexbox untuk tampilan yang responsif.

[4] State Management (P04): Menggunakan useState untuk mengelola input teks dan array daftar tugas.

[5] Conditional Rendering: Implementasi perubahan gaya visual (coret teks) pada tugas yang selesai dan tampilan khusus saat list kosong.

[6] Form Input & Validasi (P05): Menggunakan TextInput dengan KeyboardAvoidingView serta validasi Alert jika input kosong.

[7] List Dinamis (P06): Menampilkan daftar tugas menggunakan FlatList dengan keyExtractor dan ListEmptyComponent.

[8] Fitur CRUD:

Add: Berhasil menambah tugas baru ke dalam list.

Delete: Berhasil menghapus tugas tertentu dari list.

[x] Fitur Bonus:

Mark as Done: Menandai tugas selesai dengan menekan item.

Prioritas Task: Indikator warna berbeda (Tinggi/Sedang/Rendah) untuk setiap tugas.

Task Counter: Menampilkan jumlah tugas yang selesai dari total tugas yang ada.

UI/UX Professional: Tema Midnight Dark Mode dengan desain kartu yang rapi dan bayangan (shadow).
